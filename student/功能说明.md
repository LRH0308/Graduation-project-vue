# 毕业设计管理系统 - 学生端

## 项目概述

基于 Vue3 + Element Plus 开发的毕业设计管理系统学生端，涵盖毕业设计全流程管理。

## 技术栈

- **框架**: Vue 3.5.29
- **UI 库**: Element Plus 2.13.5
- **路由**: Vue Router 5.0.3
- **状态管理**: Pinia 3.0.4
- **HTTP 请求**: Axios 1.13.6
- **构建工具**: Vite 7.3.1
- **语言**: JavaScript (ES6+)

## 功能模块

### 1. 首页 (Home)
- 系统欢迎界面
- 快捷功能入口（课题选题、任务书、开题报告、中期检查、论文提交、答辩管理）
- 重要通知时间轴

**文件路径**: `src/views/home/Home.vue`

### 2. 课题管理

#### 2.1 课题列表 (SubjectList)
- 搜索功能：按课题名称、导师姓名、课题类型筛选
- 课题列表展示（分页）
- 课题详情查看
- 在线选题功能

**文件路径**: `src/views/subject/SubjectList.vue`

#### 2.2 我的课题 (MySubject)
- 已选课题信息展示
- 课题详细信息（简介、研究内容、预期成果）
- 当前进度可视化（步骤条）

**文件路径**: `src/views/subject/MySubject.vue`

### 3. 任务书查看 (TaskBook)
- 查看导师下达的任务书
- 任务书详细内容：
  - 毕业设计题目
  - 主要内容和要求
  - 技术指标
  - 进度安排
  - 预期成果
  - 参考文献
- 审核状态和意见
- 附件下载

**文件路径**: `src/views/task/TaskBook.vue`

### 4. 开题报告提交 (ProposalSubmit)
- 在线填写开题报告：
  - 研究背景
  - 研究现状
  - 研究内容
  - 技术路线
  - 创新点
  - 进度安排
  - 参考文献
- 报告文档上传
- 提交状态跟踪
- 导师审核意见查看
- 支持重新提交（审核驳回后）

**文件路径**: `src/views/proposal/ProposalSubmit.vue`

### 5. 中期成果提交 (MidSubmit)
- 中期报告填写：
  - 已完成工作
  - 阶段性成果
  - 遇到的问题
  - 后续计划
- 附件材料上传
- 导师评分查看
- 审核意见查看
- 支持重新提交

**文件路径**: `src/views/midCheck/MidSubmit.vue`

### 6. 论文终稿提交 (PaperSubmit)
- 论文信息填写：
  - 论文标题
  - 关键词
  - 摘要
  - 备注
- 论文文件上传
- 提交状态跟踪
- 导师审核状态和意见
- 支持重新提交

**文件路径**: `src/views/paper/PaperSubmit.vue`

### 7. 答辩管理

#### 7.1 答辩安排 (DefenseArrangement)
- 答辩基本信息（批次、日期、时间、地点）
- 答辩小组成员信息
- 答辩要求
- 答辩流程说明
- 注意事项

**文件路径**: `src/views/defense/DefenseArrangement.vue`

#### 7.2 答辩成绩 (DefenseResult)
- 答辩成绩详情
  - 各评分项目得分（论文质量、自述表现、问题回答、系统演示）
  - 加权计算总成绩
  - 成绩等级（优秀、良好、中等、及格、不及格）
- 评委组评语
- 答辩问题记录

**文件路径**: `src/views/defense/DefenseResult.vue`

### 8. 过程指导记录 (GuideRecord)
- 指导记录时间轴展示
- 记录详情：
  - 指导标题
  - 指导方式（面对面/线上）
  - 指导内容
  - 布置任务
  - 指导教师
- 按日期筛选
- 分页查看

**文件路径**: `src/views/record/GuideRecord.vue`

## 公共组件

### UploadFile.vue
- 通用文件上传组件
- 支持文件类型：doc, docx, pdf, zip, rar
- 文件大小限制：50MB
- 上传成功/失败回调
- 文件数量限制

**文件路径**: `src/components/UploadFile.vue`

### Layout.vue
- 系统布局组件（侧边栏 + 头部）
- 侧边栏菜单导航
- 用户信息展示
- 下拉菜单

**文件路径**: `src/components/Layout.vue`

## 代码规范

### 命名规范
- 组件名：PascalCase（大驼峰）
- 变量名：camelCase（小驼峰）
- 常量名：UPPER_SNAKE_CASE（大写蛇形）
- 文件名：PascalCase.vue

### 注释规范
- 所有代码均包含中文注释
- 组件顶部说明功能
- 关键逻辑添加行内注释

### 开发规范
- 使用 Vue 3 Composition API（setup 语法糖）
- 不使用 TypeScript
- 数据通过接口获取（当前为示例数据）
- 统一使用 Element Plus 组件库
